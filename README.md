# Sentiment Analysis: Classical ML vs Transformer Models

## Research Question
How well can classical machine learning and transformer-based models classify sentiment in text?

## Methods
- Logistic Regression with TF-IDF
- Transformer model (DistilBERT)

## Results
- Logistic Regression Accuracy: 0.822
- Transformer model shows strong confidence and better contextual understanding in predictions

## Interpretation
Transformer models capture deeper language patterns than classical methods.

## Limitations
- Small dataset subset
- No hyperparameter tuning

## Future Work
- Fine-tune transformer models
- Use larger datasets
